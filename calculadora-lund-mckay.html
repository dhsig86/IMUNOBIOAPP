Calculadora Lund-Mackay

Calcule a pontuação de opacificação sinusal com base no sistema Lund-Mackay.

A calculadora Lund-Mackay é usada para avaliar a opacificação dos seios paranasais com base em exames de imagem. Para cada seio, use o controle deslizante para selecionar o nível de opacificação:

  • 0: Sem opacificação
  • 1: Opacificação parcial
  • 2: Opacificação total

O escore total varia de 0 a 24, sendo que valores mais altos indicam maior obstrução e inflamação.
